[[!meta date="Mon, 14 Dec 2012 23:45:00 +0200"]] [[!pagetemplate template="news.tmpl"]] [[!meta title="Call for testing: 0.16~rc1"]]

You can help Tails! The first release candidate for the upcoming version 0.16 is out. Please test it and see if all works for you.

[[!toc levels=1]]

How to test Tails 0.16~rc1?

  1. Keep in mind that this is a test image. We have made sure that it is not broken in an obvious way, but it might still contain undiscovered issues.

  2. Download the ISO image and its signature:

    Tails 0.16~rc1 ISO image

    Tails 0.16~rc1 signature

  3. Verify the ISO image.

  4. Have a look at the list of known issues of this release and the list of [[longstanding known issues|support/known_issues]].

  5. Test wildly!

If you find anything that is not working as it should, please [[report to us|doc/first_steps/bug_reporting]]! Bonus points if you check that it is not a known issue of this release or a [[longstanding known issue|support/known_issues]].

What's new since 0.15?

  • Minor improvements
    • Replace the too-easy-to-misclick shutdown button with a better "Shutdown Helper" Gnome applet.
    • Display ~/Persistent in GNOME Places and GtkFileChooser if it is mounted.
    • Set Unsafe Browser's window title to "Unsafe Browser".
    • Install ekeyd to support the EntropyKey.
    • Install font for Sinhala.
    • Update Poedit to 1.5.4.
    • Kill Vidalia when restarting Tor.
    • Hide the persistence setup launchers in kiosk mode.
  • Bugfixes
    • Disable IPv6 on all network interfaces.
    • Don't look for persistence volume on internal disks.
    • Fix SCIM in the autostarted web browser
    • Talk of DVD, not of CD, in the shutdown messages.
    • Make tordate work in bridge mode with an incorrect clock.
  • iceweasel
    • Set iceweasel homepage to the news section on the Tails website.
    • Hide the iceweasel add-on bar by default.
    • Don't hide the AdBlock-Plus button in the add-on bar anymore.
    • Do not install a placeholder (fake) FireGPG iceweasel extension anymore.
    • Don't install xul-ext-monkeysphere anymore.
  • Build system
    • Install the "standard" task with tasksel regardless of the build system being used.
    • Install p7zip-full explicitly.
    • Remove useless pinning of libvpx0.
    • Remove config/chroot_local-packages/ from .gitignore.
    • Make it clearer what can and cannot be done in terms of local packages.

Known issues in 0.16~rc1

None.
